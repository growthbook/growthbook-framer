# GrowthBook Framer Plugin

This is a template for using the Framer Plugin API in a TypeScript project.

## Quickstart

First, run the development server:

```bash
pnpm dev
```

## Structure

This repo contains code for two projects:

1. GrowthBook Framer Pluging | `src/**`
2. GrowthBook Framer Code Component `FramerComponent.tsx`



Learn more: https://www.framer.com/developers/plugins/introduction
